## 二、多视角 Agent 推演

> 模拟 5 个核心利益相关方视角，每个 Agent 基于其目标函数和信息集做出行为推演。

### 2.1 🏛️ Agent A：刚果金政府

**目标**：控制疫情、维持政权稳定、获取国际援助

**信息集**：Ituri 三省疫情数据、冲突地区安全情报、财政预算约束

**推演行为**：
- **短期（1-2 周）**：配合 WHO 宣布国家卫生紧急状态，激活 IMS 指挥系统。但因 Ituri 持续冲突，安保走廊谈判成为第一优先级——没有安全保障，接触追踪和疫苗接种（即使有疫苗）无法开展。
- **中期（1-3 月）**：面临两难——国际社会要求加强监测和限制人口流动，但 Ituri 1.9M 难民 + 每月上万新流离失所者意味着封锁不现实。最可能的策略是重点防控三个核心 HZ（Mongbwalu/Rwampara/Bunia），外围采用症状监测而非强制隔离。
- **风险点**：军队介入防疫可能引发社区抵触（历史上发生过）；资金挪用风险；选举周期可能使决策短期化。

### 2.2 🌍 Agent B：WHO / 国际社会

**目标**：遏制疫情全球扩散、协调多国响应、推动疫苗/药物研发

**信息集**：全球实验室网络、IHR 框架、各国入境政策、研发管线

**推演行为**：
- **短期**：PHEIC 机制已启动。最大挑战是 **BDBV 无疫苗无特效药**——这是与 2014-2016 西非埃博拉和 2018-2020 DRC 埃博拉的本质区别。响应完全依赖非药物干预（监测+隔离+安全埋葬+社区动员）。
- **中期**：加速 BDBV 疫苗和药物临床试验。但疫苗从零到可用至少 6-12 个月。在这期间，疫情规模将决定国际社会的风险承受力。
- **关键判断**：WHO 评估 DRC 国家风险 "非常高"、区域风险 "高"、全球风险 "低"——这意味着现阶段不建议旅行限制或边境关闭。

### 2.3 👥 Agent C：Ituri 社区居民

**目标**：生存、保护家人、维持生计

**信息集**：谣言与社交媒体、传统信仰、过往埃博拉经历的记忆、对政府和外来者的信任度

**推演行为**：
- **信任赤字**：Ituri 长期冲突导致社区对政府和国际组织信任度极低。历史上埃博拉响应中的强制隔离和安全埋葬曾引发暴力抵抗。
- **经济压力**：Mongbwalu 是矿区，居民以采矿为生。限制流动 = 失去收入来源 = 更可能在症状轻微时隐瞒、继续工作。
- **信息战**：WHO 已启动 UNICEF 数字平台 (U-Report/I-Hear-You) 进行社会倾听，但谣言传播速度 > 官方信息纠正速度。60% 病例为女性（家庭护理者角色），意味着社区动员必须以女性为核心。
- **最坏情景**：社区将防疫人员视为威胁 → 隐藏病例 → 传统葬礼加速传播 → 疫情失控。

### 2.4 🚨 Agent D：邻国（乌干达 / 南苏丹 / 卢旺达）

**目标**：防止本土传播、维持边境贸易、保护医疗系统

**信息集**：跨境疫情数据、口岸监测能力、国内 ICU 床位

**推演行为**：
- **乌干达**：已有 2 例输入，但目前无本地传播。经历过 2025 年苏丹埃博拉和多次疫情，响应经验丰富。最大风险是 Kampala（首都，200 万人口）的输入病例引发的城市传播。
- **南苏丹**：医疗系统极其脆弱，一旦输入病例后果严重。Ituri 与南苏丹接壤，但边境管控薄弱。
- **区域协调**：IOM 在支持口岸监测，但"非正式过境点和弱警报管理仍是重大缺口"（WHO 原话）。跨境接触者追踪需要双边协议，而这在冲突地区几乎不可能做到。

### 2.5 💰 Agent E：全球市场 / 地缘政治

**目标**：评估经济影响、供应链风险、政治稳定

**信息集**：钴矿供应链（DRC 占全球 70%）、保险市场、旅行行业

**推演行为**：
- **矿产供应链**：Ituri 和 North Kivu 是重要矿区（金、钴、钽）。疫情可能导致矿山停工或出口受阻 → 全球钴价波动。
- **旅行保险**：PHEIC 触发后，保险公司可能提高前往东非的保费或增加排除条款。
- **地缘政治**：DRC 是中国在非洲的重要矿产来源国。中国可能通过双边渠道提供医疗援助以保护供应链，参考 2014 年西非埃博拉期间的中国援建治疗中心模式。

## 三、推演轨迹：三种情景

### 情景 A：快速遏制（概率 25%）

- **条件**：核心三 HZ 在 4 周内实现有效隔离 + 接触追踪率 > 70% + 社区接受度高
- **结果**：累计确诊 200-400 例，6-8 周内宣布疫情结束
- **前提**：Ituri 冲突出现意外停火或安保走廊快速建立

### 情景 B：渐进控制（概率 50%）

- **条件**：三省持续报告新病例，但增长率逐步下降；接触追踪率从 21% 提升至 50%+
- **结果**：累计确诊 800-2000 例，持续 3-6 个月，偶发跨境簇
- **这是最可能的情景**，符合 DRC 历史上大多数埃博拉疫情的模式

### 情景 C：长期扩散（概率 25%）

- **条件**：冲突升级导致响应中断 + 社区抵抗 + 城市传播（Bunia/Kampala）
- **结果**：累计确诊 > 5000 例，多国蔓延，持续 > 12 个月
- **触发信号**：乌干达出现本地传播链、南苏丹报告首例、Kinshasa 出现病例
- **区别于情景 B 的关键**：城市传播一旦建立，R0 将显著升高，控制成本指数级增加

## 五、教训与启示

### 5.1 对全球卫生安全

1. **BDBV 暴露了检测盲区**：GeneXpert（最广泛使用的现场检测平台）无法检测 Bundibugyo 病毒。这次疫情中，20 份样本在省级实验室用 Ebola Xpert 检测均为阴性——延误诊断 10 天。全球需要更新 Filovirus 检测 Panel。
2. **无疫苗的埃博拉才是真正的埃博拉**：过去十年我们习惯了"有疫苗的埃博拉"（rVSV-ZEBOV）。BDBV 提醒世界：Filovirus 有 6 个种，只有一个有疫苗。

### 5.2 对冲突地区防疫

3. **安保走廊是第一响应基础设施**：在 Ituri，不是缺 PPE 也不是缺培训——是缺乏安全通路。WHO 正在协调直升机和装甲车用于响应——这在传统防疫中是罕见的。
4. **"4 周检测缺口"是系统性失败**：首例 4.24 发病，5.14 才确诊。20 天的延迟中，病毒在社区和医院中自由传播。这不仅仅是实验室问题——是临床怀疑指数低 + 几种共循环病毒（虫媒病毒、流感样疾病）掩盖了信号。

### 5.3 方法论反思

5. **MiroFish 方法的验证**：本分析使用的"知识图谱 → 多 Agent 推演 → 情景分析"框架，在本次真实疫情分析中表现出良好的结构性和可操作性。关键成功因素：
   - 高质量种子数据（WHO DON 是权威一手来源）
   - 多视角 Agent 模拟覆盖了通常被忽略的社区和邻国视角
   - 风险节点表格可以作为动态更新仪表盘的基础

## 六、校验记录

### R1：数据一致性校验 ✅

- 对比 DON602（5.16）和 DON603（5.21）数据：5 天内疑似病例从 246 → 746（+203%），符合指数增长模式
- 确认病例从 8 → 85，与 4.24 首例以来的 4 周时间窗口匹配（潜伏期 2-21 天）
- CFR 12% 远低于历史 BVD 的 30-50%——可能原因是大量疑似病例稀释分母，同时也可能反映本次毒株毒力较低

### R2：逻辑自洽性校验 ✅

- 冲突 → 低追踪率 → 加速传播 的逻辑链得到数据支持（21% 追踪率 → 1603 名接触者中仅 1 名被识别为疑似 → 大量社区死亡可能未被发现）
- 无疫苗 → 完全依赖非药物干预 → 在冲突地区非药物干预受限 → 控制难度大，逻辑自洽
- 矿区起源假设与 Mongbwalu 高流量 + 人口高流动性一致

### R3：多源交叉验证 ✅

- WHO DON602/603 数据一致
- 时间线无内部矛盾
- PHEIC 声明与国际法程序一致
- 历史 BVD 疫情数据（2007 乌干达、2012 DRC）与本次 CFR 差异已标注并解释
