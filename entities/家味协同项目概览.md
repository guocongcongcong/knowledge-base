# 家味协同 (JiaWei)

家庭点菜协作 Web App —— React + Spring Boot + MySQL 全栈菜谱/周计划应用。

## 技术栈

| 层 | 技术 | 端口 |
|---|---|---|
| 前端 | React (Vite) + Zustand + React Router | :5173 |
| 后端 | Spring Boot 3.4 + JPA | :8080 |
| 数据库 | MySQL 8.4 (Docker) | :3307 |
| AI | Python ai-server.py (DeepSeek v4-flash) | :3456 |

## 数据规模

- 357 道中餐菜谱（来源 HowToCook 仓库）
- ~1046 个去重食材
- ~3100 条菜谱-食材关联
- 7 张数据表

## 页面路由

| 路由 | 页面 | 功能 |
|------|------|------|
| / | HomePage | 分类入口、搜索、季节性、本周计划、最近浏览 |
| /category/:cat | CategoryPage | 分类列表、难度筛选、收藏 |
| /recipe/:slug | RecipeDetail | 菜谱详情、步骤、食材缩放、加入计划/菜篮 |
| /add-recipe | AddRecipe | 自定义菜谱表单 |
| /ai-import | AiImport | URL → AI 提取菜谱 → 保存 |
| /basket | BasketPage | 按超市货道分类的购物清单 |
| /plans | PlanPage | 创建/查看周计划，点单统计 |
| /plan/:id | PlanDetail | 计划明细，按用户分单，锁定/解锁 |
| /favorites | FavoritesPage | 收藏菜谱列表 |
| /auth | AuthPage | 昵称编辑、点单历史 |

## 设计系统

- 品牌色: #F57C00 (橙色)
- 背景: #FFF8F0 (暖白) / #1a1a2e (暗色)
- CSS 变量 + 暗色主题切换
- 响应式移动端优先

## 关键架构决策

- RESTful API (`/api/...`)，非 PostgREST
- JPA entity 必须用 `Integer id`（对应 MySQL INT AUTO_INCREMENT）
- 菜谱步骤存为 JSON 字符串，前端需要 `JSON.parse`
- 食材导入用 pymysql 直连 MySQL（不用 API）
- AI 导入用 DeepSeek，支持 B站视频 URL
- macOS 休眠会杀掉 Spring Boot 和 Vite
