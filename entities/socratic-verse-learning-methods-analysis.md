## 执行摘要

今天（2026-06-22）锅釨用 V1 苏格拉底模式学完了入门篇核心链全部 5 篇教材。但 V1-V8 共 8 种学习方法中，实际只用过 V1 和 V7 两种——其余 6 种定义在 `prompts/system-prompts.md` 但从未被执行。此外，教师轮值逻辑冲突、教材 order 编号错位、复习机制缺失等问题影响教学效果。

**核心结论：8 种方法不是"可选菜单"——它们是一个生命周期。V1 是种子，V5 是收割，V2 是检验，V7 是巩固，V3 是保鲜，V4 是惊喜，V6 是体检，V8 是管家。**

## 一、V1-V8 各模式调用时机（Hermes + Claude 合并分析）

### 合并共识（两人的分析完全一致）

| 模式 | 最佳时机 | 黄金搭档 | 不适合 |
|:---:|------|------|------|
| V1 苏格拉底 | 第一次学新概念，精力充沛 | V1 → V5 学完立刻复述 | 疲劳时、纯事实性知识 |
| V2 费曼 | V1 课后 24h，检验理解深度 | V5 → V2 发现漏洞后填补 | 刚接触还没消化的概念 |
| V3 Anki | 每天 08:30，对抗遗忘曲线 | **应持续运行**，与所有模式并行 | 概念太少时（<10 个） |
| V4 骰子 | 动力低谷、决策瘫痪时 | 每周一次，作为"惊喜复习" | 有明确学习目标时 |
| V5 合书复述 | 学完 1h 内，黄金提取窗口 | V1 → V5 是最强搭配 | 超过 48h 后 |
| V6 交错魔鬼 | 多科全学完后，找概念边界 | **入门阶段禁用**，进阶篇启用 | 入门阶段、疲劳时 |
| V7 对子互教 | V1 课后 72h，需要换视角 | V2 → V7 检验通过后巩固 | 概念还没独立理解时 |
| V8 自适应 | V1-V7 全部验证后作为默认 | 当前不应启用 | 依赖信号还没收集够时 |

### Claude Code 独到发现

1. **V2 费曼是 V1 苏格拉底的"镜像"** — 苏格拉底是老师追问学生，费曼是学生教老师。两者结合才能同时覆盖"被追问"和"主动讲"两个方向。目前只有前者。
2. **V4 骰子依赖题库** — prompt 说"掷骰子（1-20）抽出对应问题"，但系统中没有显式的题库文件。需要基于已学教材自动生成。
3. **V7 的实际效果有数据支撑** — 2026-06-16 的 V7 课中，学习者主动创造了"削铅笔"类比，是对子互教模式效果的直接证据。
4. **V8 不应急于启用** — 在 V1-V7 各自的效果还没验证完之前，V8 的调度决策没有依据。

## 二、当前系统 5 大问题（合并排名）

### 🔴 问题 1：V2-V8 大面积未生效（Hermes + Claude 共识）

8 种方法宣传，实际只用过 V1（5 次）和 V7（1 次）。根因：
- 没有"课后自动触发其他方法"的机制
- 用户必须手动指定版本号
- V3/V4 依赖的题库从未建立
- 没有学习方法调度表

### 🔴 问题 2：复习机制完全缺失（Claude 明确指出，Hermes 确认）

核心链 5 篇全部标 ✅，但没有任何后续复习计划。2026-06-16 学的理解Agent 到今天已 6 天，按艾宾浩斯遗忘曲线已遗忘约 50%。`progress.md` 只有"学了什么"，没有"什么时候该复习什么"。

### 🔴 问题 3：教师轮值混乱（两人共识）

- `system.md` 说轮值顺序：三月七→甘雨→刻晴
- 实际：教材内容跨度过大，一个老师硬教不擅长的内容效果差
- 驾驭工程先由三月七开始后由刻晴重讲——浪费一轮

### 🟡 问题 4：教材 order 编号错位（已被 materials-map 标记 P0，未执行）

旧 order 编号让三篇权威参考文献排在核心教材前面。`materials-map.md` 已标记 P0 修复但未执行。

### 🟡 问题 5：Context 扫读 vs 学习混淆（双方分析一致）

关联图谱分析时通读了全文，系统误以为"读过=学过"，直接跳过教学。

## 五、理想的一节课生命周期

以"理解Agent"为例，展示所有方法如何协同：

```
Day 0  19:00  V1 苏格拉底 — 学新概念（45-90min）
Day 0  20:00  V5 合书复述 — 立刻合上书复述（15min）
Day 0  21:00  自动生成 5 道 Anki 推理题 → 写入 anki_deck.md
Day 1  19:00  V2 费曼 — 教一个"高中生"（20min）
Day 3  19:00  V7 对子互教 — 和 AI 同学互讲（30min）
Day 7  08:30  V3 Anki — 间隔重复推送首次出现
Day 14 08:30  V3 Anki — 根据回答质量决定是否再出现
Day 14 20:00  V4 骰子 — 随机抽检（如果抽到）
（核心链 5 篇全部学完后）
Day N  19:00  V6 交错魔鬼 — 5 篇概念混取（60min）
```

**关键原则**：V1 是种子，V5 是收割，V2 是检验，V7 是巩固，V3 是保鲜，V4 是惊喜，V6 是体检，V8 是管家。
