# 食气

## 概述

「食气」有双重谱系：一脉源自先秦道家与道教的**养生修仙传统**，一脉演化为当代玄幻小说的**修炼体系设定**。两者同源异流，内核截然不同——前者是修心手段，后者是变强燃料。

---

## 一、传统文化中的「食气」

### 起源 —— 先秦道家

最早见于《庄子·逍遥游》：

> 「藐姑射之山，有神人居焉，肌肤若冰雪，绰约若处子，**不食五谷，吸风饮露**。」

——不吃五谷，靠吸纳风露（天地清气）存活，被视为「食气」思想源头。

《楚辞·远游》：「餐六气而饮沆瀣兮，漱正阳而含朝霞」，屈原笔下的仙人食用的是「六气」——天地间六种精纯之气。

### 六气详解

源自东汉王逸《楚辞章句》引《陵阳子明经》，六气分为**四时之气 + 天地之气**：

| 六气 | 时辰 | 方位 | 五行 | 形态 | 主要功效 |
|------|------|------|------|------|----------|
| **朝霞** | 平旦（日出） | 东方 | 木 | 日初出，赤黄之气 | 主**生发**。补肝气、升阳气、启神明。采之可令人精神振作，目明耳聪。对应春气，万物萌动之机。 |
| **正阳** | 日中（正午） | 南方 | 火 | 日正中，阳气最盛 | 主**长养**。补心气、壮元阳、旺血脉。采之可驱寒湿、暖四肢、强心志。一日阳气之巅，不宜多采以防火亢。 |
| **飞泉** | 日入（黄昏） | 西方 | 金 | 日落之气，又名「沦阴」 | 主**收敛**。补肺气、润皮毛、收魂魄。采之可宁心定志、安神助眠。对应秋气，由收转藏之节点。 |
| **沆瀣** | 夜半（子时） | 北方 | 水 | 夜半清露之气 | 主**闭藏**。补肾气、填精髓、养元阴。采之可强骨髓、乌须发、固根本。子时一阳生，阴极阳生之秘在此。 |
| **天玄** | 不分时辰 | 天 | —— | 天色，玄气 | 主**清灵**。补元神、开智慧、通灵觉。天之气轻清上浮，养人之神，为思虑清明之源。 |
| **地黄** | 不分时辰 | 地 | —— | 地色，黄气 | 主**厚载**。补脾胃、充肌肉、实形体。地之气浊重下沉，养人之形，为体力耐力之本。 |

#### 六气的整体逻辑

```
朝霞（木·肝·生）→ 正阳（火·心·长）→ 飞泉（金·肺·收）→ 沆瀣（水·肾·藏）
                      ↕ 天地总摄 ↕
                 天玄（神）＋ 地黄（形）
```

- **四时四气**：按一日循环对应四季，构成人体小周天。朝采霞、午采阳、暮采泉、夜采瀣，一昼夜完成一轮精气轮转。
- **天地二气**：不拘时辰，玄气养神、黄气养形，形神兼备方为全功。
- 缺一行则偏颇：只采朝霞不采沆瀣则阳亢阴虚，只采正阳不采飞泉则火旺金弱。

《抱朴子》的印证：「餐朝霞之沆瀣，吸玄黄之醇精」——朝霞、沆瀣、玄、黄，四者并举。

### 其他可食之气（六气之外）

六气是「时辰之气」，除此之外道家还有多套食气对象，可按采食目标分类：

#### 一、日月星辰之气

| 气名 | 来源 | 功用 |
|------|------|------|
| **日精** | 太阳光芒，正午最佳 | 补元阳、驱阴邪、炼目力。陶弘景《养性延命录》有「服日芒法」 |
| **月华** | 满月精光，夜半最佳 | 补元阴、养魂魄、润容颜。上清派有「服月精法」，存思月华入丹田 |
| **北斗气** | 北斗七星降下精华 | 通神明、延寿命。正一道醮仪中北斗崇拜极重，《北斗本命延生经》专论 |
| **三光气** | 日月星合称 | 统补一身精气。存思日月星光芒汇入百会 |

#### 二、五行/五方之气

| 气名 | 方位 | 五行 | 色 | 对应脏 |
|------|------|------|------|------|
| **青牙** | 东方 | 木 | 青 | 肝 |
| **朱丹** | 南方 | 火 | 赤 | 心 |
| **黄庭** | 中央 | 土 | 黄 | 脾 |
| **素华** | 西方 | 金 | 白 | 肺 |
| **玄冥** | 北方 | 水 | 黑 | 肾 |

> 《黄庭经》称「五芽」—— 春天食青芽之气、夏食朱丹、秋食素华、冬食玄冥、四季末各十八日食黄庭。按季节单采一气补对应脏腑，也可五气同采以求平衡。

#### 三、虚空/混沌之气

| 气名 | 含义 |
|------|------|
| **太和之气** | 天地未分时的原始和谐之气，采之可「和阴阳、通天地」。在宋元内丹学中被视为最高级别的食气对象 |
| **先天一气** | 宇宙创生之前的第一股能量。内丹学的终极追求——「得先天一气，便可超凡入圣」 |

#### 四、草木山川之气

| 气名 | 含义 |
|------|------|
| **云气** | 山间云雾之气，《庄子》「吸风饮露」的风即云气 |
| **石髓玉液** | 山石渗出的矿物精华，被视为液体形态的气 |
| **草木清气** | 松柏灵芝之气尤贵。《抱朴子》：「食松柏之实，可延年」——松柏四季长青，其气最纯 |

> **小结**：六气是入门，日月精华是高阶，先天一气是终点。从有形的日月到无形的混沌，食气的对象越精纯，修行者的境界越高。

### 食气之法（具体操作）

食气不是简单的深呼吸，而是一套精密的身心技术。以下是几种核心方法：

#### 一、咽气法（基础入门）

最朴素的食气术，要领是 **吸→闭→咽→导**：

```
1. 平旦（日出前），面东而坐，叩齿三十六通
2. 鼻引清气入满口，闭息至极限
3. 分三次缓缓咽下，观想气入丹田
4. 以意引气循行周身
```

唐代《嵩山太无先生气经》详载此法，称「凡服气，取子后午前，生气时也」。

#### 二、六字诀（以声引气）

孙思邈《千金方》传下的六字吐纳法，以六个发音对应六脏：

| 字诀 | 发音 | 对应脏 | 功效 |
|------|------|------|------|
| 嘘 xū | 撮口呼出 | 肝 | 去肝火、明目 |
| 呵 hē | 开口呼出 | 心 | 降心火、安神 |
| 呼 hū | 圆唇呼出 | 脾 | 消食滞、化湿 |
| 呬 sī | 齿缝出气 | 肺 | 清肺热、利咽 |
| 吹 chuī | 撮唇出气 | 肾 | 补肾气、强腰 |
| 嘻 xī | 颊车开合 | 三焦 | 调三焦、理气机 |

用法：鼻吸清气 → 口吐字诀浊气（不出声或微出声）→ 各六次为一个循环。

#### 三、存思食气法（以意引气）

上清派的核心技术。不是单纯呼吸，而是**在想象中「看见」气进入身体**：

> 存思东方青气如丝如雾，自远而来，由鼻入喉，循经下注肝脏，肝中一片青莹……

- **服日芒法**：存思日中有一九芒赤车，光芒万丈，吞此光芒入喉、入心、入丹田
- **服月精法**：存思月中玉兔捣药，莹白之精如水银泻地，从顶门灌入脊椎
- **五芽法**：按季节存思五方五色之气各入对应脏腑

存思法的关键在于「意到气到」——心念聚焦之处，气自然汇聚。

#### 四、胎息法（高阶闭气）

葛洪《抱朴子》：「得胎息者，能不以鼻口嘘吸，如在胞胎之中。」

- 目标：呼吸微细至「以踵呼吸」（感觉气息直达脚后跟），最后进入几乎无呼吸的状态
- 训练：从闭息九十息起步，逐步增加到三百息；鼻前放羽毛，羽毛不动方为入门
- 原理：关闭外呼吸，启动内呼吸——直接以皮肤毛孔吸纳天地之气

#### 五、食气辟谷的实操阶梯

从初学到登堂入室，食气辟谷有严格的次第：

```
第一阶：减食 → 日中一食，只吃五谷粥糜
第二阶：代食 → 以药饵（黄精、松仁、胡麻）代替谷物
第三阶：服气开始 → 每日定时咽气，辅以少许药饵
第四阶：纯粹服气 → 不食任何有形食物，单靠咽气存思
第五阶：胎息 → 呼吸近乎停止，毛孔食气
```

> 关键提醒：辟谷不是绝食。如果没有服气基础就强行断食，是「自戕」而非「修仙」。历代道经反复警告：**未通食气而先绝谷者，必伤脏腑**。

### 道教体系中的系统化发展

在道教内丹学与养生术里，「食气」是「辟谷」（断绝五谷）的核心手段：

| 概念 | 含义 |
|------|------|
| **服气** | 通过吐纳（呼吸法）「吃」进天地精气，替代谷物 |
| **六气** | 朝霞、正阳、飞泉、沆瀣、天玄、地黄，不同时辰采不同的气 |
| **食气者神明而寿** | 《淮南子》经典论断：食气者比食谷者更长寿、更通灵 |
| **胎息** | 高阶食气术，模拟胎儿在母体中的呼吸方式，被认为可以「返先天」 |

**核心理念**：人的身体不只是靠物质食物供养，还可以直接吸收宇宙精气。谷物是「浊气」来源，吃得越少、越清净，越接近「真人」状态。这是修仙（追求长生）的生理基础。

## 二、玄幻小说中的「食气」

### ① 基础修炼层 —— 炼气期

绝大多数修仙/玄幻体系的起点。修炼者通过功法吸纳天地「灵气」入体，在丹田炼化，本质上是对「食气」概念的现代化包装：

> 功法 → 灵气入体 → 丹田炼化 → 实力提升

经典设定：炼气期通常分九层（或十三层），是修士打基础的阶段。

### ② 能量的层级化

传统文化只有「清气 vs 浊气」，玄幻则建立了精密的气的「鄙视链」：

```
混沌之气 > 鸿蒙紫气 > 神气 > 仙气 > 灵气 > 元气 > 普通空气
```

不同的「食气」对象决定了角色的天花板和位格。

### ③ 食气的「黑暗版本」

将「食气」从吸收自然能量扭曲为**吞噬他人的生命/修为**：

- 魔道功法：吞噬修士的精血或元婴
- 邪功：吸取凡人生命力修炼
- 主角金手指（如《凡人修仙传》韩立的噬金虫、《完美世界》的吞噬类宝术）

本质上是把「食气」从道家清修变成了弱肉强食的丛林法则。

### ④ 食气与灵食体系

不少玄幻设定「灵食」——用灵材烹制的食物，吃下去可补充灵气、突破瓶颈。这是「食气」最接地气的演变：既然修士需要灵气，那把灵气做进菜里。催生了炼丹师、灵厨等职业设定。

## 四、中医里的「气」

### 气的分类

中医将人体之气按来源、功能、循行分为五大类：

| 气名 | 来源 | 分布 | 功能 | 通俗理解 |
|------|------|------|------|----------|
| **元气** | 先天（父母肾精）+ 后天（脾胃充养） | 藏于肾，循三焦布全身 | 生命原动力，推动生长发育，温煦脏腑 | 人体的「电池」——出厂自带，后天充电 |
| **宗气** | 肺吸入清气 + 脾胃运化水谷精微 | 聚于胸中（膻中），贯心脉、行呼吸 | 助肺呼吸、助心行血、发声言语 | 心肺的「燃料」——呼吸+吃饭合成 |
| **营气** | 脾胃运化水谷精微 | 行于脉中，周流全身 | 化生血液，营养全身 | 血液里的「营养快递」 |
| **卫气** | 脾胃运化水谷精微（剽悍滑利部分） | 行于脉外，布散体表 | 温分肉、充皮肤、肥腠理、司汗孔、御外邪 | 人体的「免疫边防军」 |
| **脏腑之气** | 先天+后天综合 | 各归其脏 | 维持各脏腑生理功能（心气、肝气、脾气、肺气、肾气） | 各器官的「工作能量」 |

> 经典原文：《灵枢·决气》「上焦开发，宣五谷味，熏肤，充身，泽毛，若雾露之溉，是谓气。」

### 气的来源（怎么补充）

```
气的生成 = 先天之气（肾） + 后天之气（脾胃 + 肺）
                ↓
        元气（命门） + 水谷精微（脾） + 清气（肺）
```

**三条补气路径：**

| 路径 | 方式 | 补什么 | 具体做法 |
|------|------|------|----------|
| **补先天（肾）** | 节欲保精、药食补肾 | 元气 | 节制房劳、避免过耗；核桃、黑芝麻、山药、熟地等补肾填精 |
| **补后天·脾** | 饮食调养 | 营气、卫气 | 规律饮食、五谷为养；黄芪、党参、白术、大枣、小米、粳米健脾益气 |
| **补后天·肺** | 呼吸吐纳 | 宗气 | 深呼吸、气功（八段锦、太极拳）、洁净空气、适度有氧运动 |

> **核心原则**：脾胃为气血生化之源，肺为气之主，肾为气之根。三脏协调，气自充盈。

### 气的失去（怎么消耗）

| 耗气方式 | 机制 | 典型表现 |
|----------|------|----------|
| **过劳** | 「劳则气耗」——体力/脑力透支直接消耗元气 | 疲乏无力、少气懒言 |
| **久病** | 「邪之所凑，其气必虚」——疾病消耗正气 | 病后体虚、恢复缓慢 |
| **情志失调** | 怒则气上、喜则气缓、悲则气消、恐则气下、思则气结 | 生气后头晕（气上）、悲伤后无力（气消） |
| **饮食不节** | 过饥则气乏无源，过饱则伤脾滞气 | 饥一顿饱一顿后浑身无力 |
| **房劳过度** | 耗肾精→精不化气 | 腰膝酸软、精神萎靡 |
| **久坐伤气** | 「久卧伤气、久坐伤肉」——不动则气滞 | 越躺越累 |
| **汗出过多** | 「气随津脱」——大汗则气随之外泄 | 大汗后虚脱感 |
| **外邪侵袭** | 正邪相争消耗卫气 | 反复感冒、畏风怕冷 |

### 气虚的典型信号

- 少气懒言：说话有气无力
- 动则汗出：稍微活动就出汗
- 神疲乏力：总觉得累，睡不醒
- 面色㿠白：面无血色发白
- 舌淡胖有齿痕：舌头胖大、边缘牙印
- 脉虚无力：脉搏细弱

## 五、食气与金丹法的交互

食气是金丹法的**母亲**。从先秦食气到宋元内丹，中间经历了一场深刻的「内化」革命。

### 从外丹到内丹：食气是转折的枢纽

东汉至魏晋，主流修仙方法是**外丹**——用铅汞等矿物在炉鼎中炼制丹药，服食以求长生。但外丹毒死了太多人（唐代就有六位皇帝死于丹毒），加上铅汞原料昂贵，修道者开始反思：

> 「既然服食身外矿石是服丹，那服食身内之气不也是服丹吗？」

这个追问直接催生了**内丹术**——以身体为炉鼎，以精气神为药物，以呼吸意念为火候，在体内炼成「金丹」。

而这场革命的**技术基础**，正是从战国一路传承下来的**食气术**。

### 食气 → 内丹的演化路径

```
先秦食气（服天地之气）
  ↓
汉魏外丹盛行 + 食气继续发展
  ↓
唐代钟离权、吕洞宾《钟吕传道集》→ 将食气术语转化为内丹术语
  ↓
张伯端《悟真篇》（北宋）→ 内丹体系成熟，食气融入「筑基→炼己」阶段
  ↓
全真道（金元）→ 食气 = 内丹筑基功，是入门必修
```

### 两者的核心对应关系

食气的每一个概念，在内丹体系中都有对应的「升级版」：

| 食气概念 | 内丹转化 | 含义 |
|----------|----------|------|
| **服气** | **筑基** | 食气是内丹的准备阶段，先通过服气打通经脉、积累精气，才能正式炼丹 |
| **咽入丹田** | **采药归炉** | 食气时将气咽入丹田 → 内丹的「采药」即采精气入丹田炼化 |
| **存思观想** | **意守 / 火候** | 食气存思 → 内丹修炼中「意守丹田」「调节火候」的意念控制技术 |
| **胎息** | **大周天 / 入定** | 胎息近乎无呼吸 → 内丹高级阶段的「息停脉住」状态 |
| **六气** | **五气朝元** | 内丹术语「五气朝元」——五脏之气汇聚泥丸，正是对六气/五芽的高级化用 |
| **辟谷** | **斩赤龙 / 断白虎** | 断绝谷物 → 内丹中「马阴藏相」「斩赤龙」等断绝生理消耗的概念 |

### 内丹对食气的三大改造

**1. 从「吃外气」到「炼内气」**

外丹失败后，修道者意识到关键不在「外」：
> 食气是吃外面的天地之气 → 内丹是激活体内的先天之气

张伯端《悟真篇》：「人人自有长生药，自是迷徒枉摆抛。」——丹药不在铅汞里，也不在天地间，就在自己身体里。食气只是引子，真正的修炼是调动体内的精气。

**2. 从「单向吸收」到「循环炼化」**

食气是单向的：外面进来 → 存到丹田。内丹是循环的：

```
炼精化气 → 炼气化神 → 炼神还虚 → 炼虚合道
   ↑___________________________________|
```

每一轮循环都比上一轮更精纯。食气的「进来→存着」变成了内丹的「进来→提炼→循环→升华」。

**3. 引入火候概念**

食气只有「采不采得到」的问题，内丹则引入了精密的「火候」控制：
- **武火**：猛力呼吸 + 强意念 → 用于采药、通关
- **文火**：轻柔呼吸 + 微意念 → 用于温养、育丹
- 火候错了，轻则无效，重则「走火入魔」

食气术中的闭息、咽气、胎息，在内丹体系里被重新解读为不同阶段的「火候」操作。

### 食气在外丹中的角色（另一条线）

虽然内丹取代了外丹，但在外丹仍然盛行的魏晋隋唐，**食气是炼丹师的标配**：

- 葛洪既是食气大师又是外丹宗师，《抱朴子》内篇同时大谈服气和炼丹
- 炼丹前的斋戒清洁，核心环节就是服气辟谷——以食气净化身体，使丹药能入经入脉
- 某些外丹典籍记载：服丹后必须辅以特定食气法，否则药力不走（丹药太重，需气来引）

本质上，食气在外丹流程中的地位是**清洁工 + 引路人**：先以食气清空身体（辟谷），再以食气引导药力（引药归经）。
