# Dify 与 n8n 模板市场分析

> 昨天与 Coze 一起研究的三个 Agent 平台模板市场。Coze 见 [[Coze扣子-Agent开发平台全攻略]]。

---

## Dify

| 项目 | 详情 |
|------|------|
| 官网 | https://dify.ai |
| 模板市场 | https://dify.ai/templates |
| Cloud 版 | https://cloud.dify.ai（需注册） |

**热门模板方向：**
- Chatbot 对话机器人
- RAG 知识库问答
- 工作流自动化
- Agent 编排
- 多模型路由

**特点**：开源可自部署，模板偏开发者向，自定义程度高。

---

## n8n

| 项目 | 详情 |
|------|------|
| 官网 | https://n8n.io |
| 工作流模板库 | https://n8n.io/workflows |
| 模板数量 | **9,868 个**（持续增长） |

**模板分类：**
- 🤖 **AI**（最火）
- 📊 Marketing / Sales / Support
- 🖥️ IT Ops / Document Ops

**热门集成链路：**
```
Google Sheets → OpenAI → Telegram/Gmail → Slack/Notion/Discord → MySQL/Postgres
```

**变现亮点**：n8n 支持**创作者变现** — 提交模板可被 featured + affiliate 分成。

---

## 三平台对比

| 维度 | Coze（扣子） | Dify | n8n |
|------|------------|------|-----|
| 定位 | AI Bot 平台 | AI 应用平台 | 通用自动化 |
| 国内可用 | ✅ | ✅ | 需自部署 |
| 模板市场 | Bot 商店 + 插件商店 | 模板市场 | 工作流模板库 |
| 变现方式 | 作品社区曝光 | 模板发布 | Featured + Affiliate 分成 |
| 代码门槛 | 无 | 低-中 | 低-中 |
| 适合场景 | 对话 Bot | RAG + Agent | 自动化流程 |
