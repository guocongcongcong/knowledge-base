# Obsidian 创作全流程（卡片笔记写作法）

基于《卡片笔记写作法》自创的 Obsidian 创作全流程，将碎片知识和视频嵌入认知网络，形成飞轮效应。^[raw/transcripts/douyin-obsidian-creation-workflow.md]

## 三阶段流程

### 阶段一：知识记录

**核心方法**：记录时就在脑中编织思维导图。

将每条碎片知识放入四个维度：
1. **是什么** — 定义和事实
2. **为什么** — 原因和原理
3. **怎么做** — 操作步骤
4. **总结升华** — 个人见解和连接

工具：vivo 原子笔记（支持电脑手机互通）+ 小v圈搜（截图工具）

### 阶段二：笔记整理

- **手动方式**：画思维导图，梳理知识结构
- **AI 辅助**：使用 AI 工具自动整理笔记
- **同步**：原子笔记实现电脑和手机间内容互相同步
- Obsidian 作为最终知识管理枢纽

### 阶段三：视频制作

1. **拍摄时想象成品**：拍摄前脑中已有最终效果
2. **素材对齐文案**：拍摄后放在对应文案后面
3. **优化文案结构**：调整叙事逻辑
4. **口播 + 剪辑**：最后完成视频输出

## 飞轮效应

```
碎片知识 → 四维记录 → 思维导图整理 → AI辅助加工
    ↑                                        ↓
    ←———— 视频成品 ←———— 文案优化 ←———— 
```

每次创作都在丰富认知网络，下一次创作会更快更好。

## 关键工具链

| 环节 | 工具 |
|------|------|
| 碎片捕捉 | vivo 原子笔记、小v圈搜 |
| 知识管理 | Obsidian |
| 笔记整理 | AI 工具 / 手动思维导图 |
| 视频制作 | 拍摄 + 剪辑软件 |
