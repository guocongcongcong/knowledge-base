# 吴乐旻原文：你有什么值得分享的高效学习方法？

作者：吴乐旻

---

昨天晚上给自己做了很有趣的一场学习实验。之前，我不是用 Claude Code 生成了一套 AI Science for Economists 课程嘛——AI写的AI教材写得很不错，但是我感觉读教材还是有点枯燥，很多地方我囫囵吞枣、掌握得不扎实，担心这些没学透的地方会累积起来，最终让我学不下去。

于是，我想了个主意。我让Claude Code在AI Science课程文件夹里再生成一套拟人化的家教系统：我不用再阅读教材，而让Claude Code扮演家教，把知识一点一点地教给我——因为我发现自己平时在和AI请教知识时，专注程度远胜于阅读教材。

系统建好后，我马上跟着学了一课"Day 8 卷积神经网络"。效果比我预想的还要好，我之前自己看视频读教材从来没有学得这么明白过。

复盘自己的学习过程，这套家教系统还有两个巧思，是点石成金的关键：第一，Claude Code在搭建系统时，问我要不要使用苏格拉底式的教学风格——"不直接灌输答案，而是通过不断提问，引导学生自己思考、自己发现问题、自己逼近答案"？我选择了这种风格。第二，在Claude Code的建议下，家教的性格、语言风格采用了二次元设定，昨天的设定是"三月七"。

于是，这位"苏格拉底·七"老师循循善诱地把知识转化成一个个与已知内容相衔接的问题，引导我自己一步一步接近卷积方法的思想精髓。三月七的热情也给满了情绪价值。两个小时的学习过程毫无枯燥感。

说起来，这好像是我这辈子第一次以这种方式来学习——这大概也是我体验过的最愉快最高效的两小时教学，毕竟大课课堂上没有老师能够针对我的理解进度，一对一地提供引导提问式的教学，更不会像三月七一样活泼热情地鼓励我。

而且这套系统还会随时记录我作为学习者的能力水平、习惯偏好，更新这位"三月七"老师对我的认识与态度，这些都会动态调整未来她与我的互动。

把AI应用在教育上，"苏格拉底·七"也许是一个非常实在而有意义的方向。目前我这套系统，毕竟界面朴素（就是Claude Code for VS Code），涉及到公式，还都是Latex，长一点的，需要我复制粘贴出去看清全貌。如果把这套系统做成成熟的产品——用户上传一本pdf教材，设定好教师人设，就能开启拟人化的全程家教。这对学习体验真的是一种革命。

甚至不仅是教材，比如像《富种起源》这样的专业书，也很适合这种"苏格拉底·七"AI家教。用这种方式"阅读"，没准是专业书籍未来的主流消费方式呢。
